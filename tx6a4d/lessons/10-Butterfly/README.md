仿生蝴蝶（遥控端）
===

这个示例实现一个略微进阶的按键逻辑：只对“按下（PressDown）”动作执行逻辑。

`loop()` 函数的执行频率为`50Hz`，即每秒钟内执行 50 次。

示例引入了局部静态变量`static bool buttonsLast[4]`，用来循环间保存上一次 loop() 执行后各个按键的状态。通过比较上一轮和本轮状态的区别，判断“按下”动作是否发生。当然，也可以同理实现当按键“弹起”（PressUp）触发动作。

在本例中，4个按键分为2组，分别用于2侧翅膀的调平/校准。校准参数并不直接作用于遥控端的原始通道发送参数，而是使用独立的通道直接发送给接收机，接收机端需要编程实现调平的逻辑。
